Installation

Comment ça marche ?

Se rendre sur le Chrome Store

Rendez-vous sur le site du Chrome Store afin de télécharger l'extension Poulpy. L'extension Poulpy est disponible ici : chrome.google.com/webstore

Télécharger l'extension

Recherchez l'application Poulpy et cliquez sur "ajoutez à chrome" l'extension sera ainsi automatiquement disponible, l'aventure commence ici.

Fonctionnalités

Des fonctionnalités pour vous accompagner.

Totalement Responsive

Poulpy est totalement responsive et ainsi elle est disponible sur toutes les appareils, tout les ordinateurs de toutes les tailles et même sur mobile.

Une navigation sécurisée

Poulpy vous informe lorsque vous visitez un site dangereux et vous oriente vers les sites sûrs.

Une navigation simplifiée

Poulpy vous simplifie votre navigation sur Internet grâce à un accès simple et intuitif à vos sites préférés.

Un couteau suisse

Poulpy regorge de nombreuses fonctionnalités, le projet est open-source : chacun peut devenir collaborateur dès maintenant !

Vous êtes connecté à de nombreux services

Vous êtes connecté à de nombreux services, vous fournissant vos mails, la météo, l'accès à différentes informations très rapidement, le store et bien d'autre !

Payer pour ce dont vous avez besoin uniquement

Poulpy coûte 5 euros par an afin de subvenir aux coûts de développement. De nombreuses fonctionnalités incroyables sont disponible sur le store pour un prix de moins de 1 euro par fonctionnalité supplémentaire.
